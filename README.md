# **在庫管理システム（Inventory Management System）**  

C#（WinForms）と PostgreSQL を使用して開発した在庫管理アプリケーションです。
商品登録、入庫・出庫、在庫数の自動更新、履歴管理など、
小規模業務を想定した基本的な在庫管理機能を実装しています。                                                                                                                                             
学習の集大成として、UI 設計、データベース設計、入力チェック、例外処理など
実務で必要となる要素を意識して構築しました。
今後も機能追加や改善を行い、より使いやすいアプリケーションを目指します。

## **機能一覧**

### **■ 商品管理**
- 商品の新規登録
- 商品情報の編集
- 商品の削除
- DataGridView による一覧表示
- 入力チェック（空欄・数値・桁数など）

### **■ 入庫処理**
- 商品の選択
- 入庫数の入力
- 在庫数の自動加算
- 入庫履歴の保存

### **■ 出庫処理**
- 商品の選択
- 出庫数の入力
- 在庫数の自動減算
- 出庫履歴の保存
- 在庫不足時のエラーチェック

### **■ 在庫管理**
- 現在の在庫数を一覧表示
- 入庫・出庫に応じたリアルタイム更新
- 商品ごとの在庫推移を管理

### **■ 履歴管理**
- 入庫・出庫の履歴一覧表示
- 日付・商品名などによるフィルタ（今後追加予定でもOK）
- 操作ログの保存

### **■ データベース関連**
- PostgreSQL によるデータ永続化
- SQL（SELECT / INSERT / UPDATE / DELETE）の実装
- フォーム間でのデータ連携
- DataStore クラスによる共通データ管理

## **技術スタック（Tech Stack）**

- **C# / WinForms**  
  デスクトップアプリケーションの開発に使用

- **.NET Framework**  
  アプリケーションの基盤として利用

- **PostgreSQL**  
  データの永続化・管理に使用

- **SQL（SELECT / INSERT / UPDATE / DELETE）**  
  CRUD 処理や履歴管理の実装に使用

- **Visual Studio**  
  開発環境として利用

- **Git / GitHub**  
  バージョン管理・ポートフォリオ公開に使用

## **工夫した点**

- **フォーム間のデータ連携を共通クラス（DataStore）で管理し、画面ごとのデータ不整合を防止**
- **DataGridView の動的更新を実装し、入庫・出庫後に即時反映されるように改善**
- **SQL を状況に応じて組み立てることで、柔軟なデータ取得と更新を実現**
- **入力チェック（空欄・数値・範囲・在庫不足など）を徹底し、誤操作によるデータ破損を防止**
- **例外処理を適切に配置し、アプリの異常終了を防ぐ安定した動作を実現**
- **UI をシンプルにまとめ、初めて使う人でも直感的に操作できるように設計**
